PROJECT_NAME: DinoSwimSim

## Description
DinoSwimSim is a web application that simulates the swimming attempts of theropod dinosaurs based on fossil footprints found in Bolivia. This project aims to educate users about the possible behaviors of these ancient creatures and how their swimming techniques might have looked given the constraints suggested by the fossil evidence.

The application allows users to input parameters such as number of feet, gait type, terrain type (soft mud vs. hard ground), and other variables to simulate how the dinosaurs would have moved in water. It uses a basic physics engine to calculate the movement patterns and presents the results through an animated visualization.

## Installation
To run DinoSwimSim locally, you need to follow these steps:

### Prerequisites
- Node.js and npm installed on your system.
- Basic knowledge of using command line/terminal.

### Steps
1. **Clone the Repository**
   
   git clone https://github.com/yourusername/dinoswimsim.git
   cd dinoswimsim
   

2. **Install Dependencies**
   
   npm install
   

3. **Run the Application**
   
   npm start
   

4. **Access the Application**
   Open your web browser and visit `http://localhost:3000`.

## Usage
Once you have the application running, you can interact with it through the user interface:

1. **Input Parameters**
   - Use the input fields to set parameters such as the number of feet, gait, and terrain type.
   - You can also adjust the speed and size of the dinosaur model.

2. **Run Simulation**
   - Click the "Start Simulation" button to run the simulation based on the input parameters.
   - The application will animate the dinosaur's movements in a simulated swimming environment.

3. **Reset and Experiment**
   - After running a simulation, use the "Reset" button to clear the current settings.
   - Feel free to experiment with different parameters to see how they affect the dinosaur's swimming attempts.

### Notes
- Ensure your internet connection is stable for loading assets and scripts.
- This app is best viewed on modern browsers like Chrome, Firefox, or Edge for optimal performance and compatibility.
